## 4. 群组协作任务执行流程图

```mermaid
flowchart TD
    Start([用户发送消息到群组]) --> A[消息路由到调度智能体]

    C -->|简单任务| D1[调度智能体直接回复]

    J1 --> K1[在界面显示子任务列表]
    J2 --> K1
    J3 --> K1

    K1 --> L[各智能体并行执行]

    L --> M1[智能体A完成]
    L --> M2[智能体B完成]
```

## 10. 消息发送与处理时序图

```mermaid
sequenceDiagram
    participant U as 用户
    participant UI as 前台界面
    participant M as 消息管理器
    participant C as 调度智能体
    participant A1 as 智能体A
    participant A2 as 智能体B
    participant DB as 数据库
    participant LLM as AI模型服务

    U->>UI: 输入消息并发送
    UI->>M: 创建消息对象
    M->>DB: 保存用户消息

    alt 单智能体对话
        M->>A1: 路由消息到智能体A
        A1->>LLM: 调用AI模型
        LLM-->>A1: 返回回复
        A1->>DB: 保存智能体回复
        A1->>UI: 推送回复消息
        UI->>U: 显示智能体回复
    else 群组协作对话
        M->>C: 路由消息到调度智能体
        C->>LLM: 分析任务需求
        LLM-->>C: 返回任务分析结果

        C->>C: 拆解为子任务
        C->>UI: 推送子任务列表
        UI->>U: 显示任务分解进度

        par 并行执行子任务
            C->>A1: 分配子任务1
            A1->>LLM: 执行任务1
            LLM-->>A1: 返回结果1
            A1->>C: 报告完成
            A1->>UI: 更新进度
        and
            C->>A2: 分配子任务2
            A2->>LLM: 执行任务2
            LLM-->>A2: 返回结果2
            A2->>C: 报告完成
            A2->>UI: 更新进度
        end

        C->>C: 汇总所有结果
        C->>LLM: 生成最终回复
        LLM-->>C: 返回最终答案
        C->>DB: 保存完整回复
        C->>UI: 推送最终回复
        UI->>U: 显示完整答案
    end
```
