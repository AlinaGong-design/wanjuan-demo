## 🎯 设计理念

### 为什么选择 40-60 分割？

1. **黄金比例**: 接近黄金分割比（0.618）
2. **视觉平衡**: 配置区不会太窄也不会太宽
3. **扩展性**: 右侧预留空间可用于：
   - 未来添加预览功能
   - 实时配置效果展示
   - 帮助文档
   - AI助手建议

### 为什么保留右侧空间？

- 保持页面平衡感
- 避免配置区过宽导致阅读困难
- 预留未来功能扩展空间
- 提供视觉呼吸感
