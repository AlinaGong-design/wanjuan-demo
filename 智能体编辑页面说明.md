# 智能体编辑页面结构说明

## 页面概览

已实现智能体编辑页面的完整UI结构，访问路径：`http://localhost:3000/#agent`

## 页面结构

### 1. 顶部工具栏

**左侧区域：**
- 返回按钮 (←)
- 智能体图标（渐变色Avatar）
- 智能体名称（可编辑输入框）
- 编辑图标按钮

**右侧区域：**
- 配置按钮
- 发布按钮
- 评测按钮
- 保存按钮
- 发布配置按钮（主按钮）

### 2. 左侧配置面板（宽度480px）

#### 2.1 智能体配置标题

#### 2.2 AI模型配置（折叠面板，默认展开）
- **模型选择**（必填*）
  - 下拉选择框
  - 选项：chat-model / GPT-4 / GPT-3.5
  - 带问号提示图标

- **提示词**
  - 多行文本输入框
  - 自适应高度（8-15行）
  - 包含示例提示词内容

#### 2.3 知识库（可展开区域）
- 标题
- 右侧：加号按钮 + 展开箭头
- 点击可展开/收起

#### 2.4 工具调用（可展开区域）
- 标题
- 右侧：加号按钮 + 展开箭头
- 点击可展开/收起

#### 2.5 文件上传
- 标题
- 右侧：问号提示图标 + 开关按钮
- 可切换启用/禁用状态

#### 2.6 交互体验（可展开区域）
- 标题
- 右侧：加号按钮 + 展开箭头
- 点击可展开/收起

### 3. 右侧预览面板

#### 3.1 预览头部
- 标题："调试预览"
- 上传图标按钮

#### 3.2 预览内容区
**智能体卡片：**
- 大尺寸头像（64px，渐变背景）
- 智能体名称（24px粗体）
- 作者信息："作者：讯瞧"
- 统计数字徽章（数字11）

**聊天输入框：**
- 圆角输入框（20px圆角）
- 占位符："有什么我可以帮助您的？"
- 发送按钮图标

**底部提示文字：**
- 灰色小字
- 内容："以上内容为AI生成，不代表开发者立场，请勿删除或修改文本标记"

#### 3.3 右侧浮动工具栏
- 设置按钮（带红点徽章）
- AI助手按钮（渐变背景）

## 技术实现

### 使用的组件
- Layout - 布局容器
- Button - 按钮
- Input / TextArea - 输入框
- Select - 下拉选择
- Switch - 开关
- Collapse / Panel - 折叠面板
- Avatar - 头像
- Space - 间距
- Divider - 分割线
- Badge - 徽章

### 使用的图标
- ArrowLeftOutlined - 返回箭头
- EditOutlined - 编辑
- SettingOutlined - 设置
- RocketOutlined - 发布/火箭
- FileTextOutlined - 文档
- SaveOutlined - 保存
- PlusOutlined - 加号
- RightOutlined - 右箭头
- QuestionCircleOutlined - 问号
- UploadOutlined - 上传
- SendOutlined - 发送

### 状态管理
```typescript
- agentName: 智能体名称
- modelType: 选择的模型类型
- promptText: 提示词内容
- fileUploadEnabled: 文件上传开关状态
- isKnowledgeExpanded: 知识库展开状态
- isToolExpanded: 工具调用展开状态
- isInteractionExpanded: 交互体验展开状态
- chatInput: 聊天输入内容
```

## 样式特点

1. **配色方案：**
   - 主色：紫色渐变 (#667eea → #764ba2)
   - 背景：灰白色 (#f5f5f5, #fafafa)
   - 文字：深灰色系 (#262626, #595959, #8c8c8c)

2. **布局特点：**
   - 左侧固定宽度480px
   - 右侧自适应剩余空间
   - 垂直滚动支持

3. **交互效果：**
   - 展开/收起动画
   - 悬停高亮效果
   - 自定义滚动条样式

4. **响应式设计：**
   - 支持不同屏幕尺寸
   - 1200px以下调整左侧宽度
   - 992px以下进一步优化
