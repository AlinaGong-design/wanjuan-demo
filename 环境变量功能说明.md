# 沙盒编辑器环境变量功能说明

## 功能概述

在沙盒编辑器中新增了常驻的**环境变量**页面，允许用户配置和管理开发环境和生产环境的环境变量。

## 访问路径

`http://localhost:3000/#skill-editor?mode=create`

## 页面结构

### 1. 顶部标签栏

#### 标签页
- **预览**：原有的技能编辑预览页面
- **环境变量**：新增的环境变量配置页面（带全局图标）
- **+按钮**：预留的添加新标签功能

#### 右侧工具栏
- 主题切换按钮（灯泡图标）
- 文件目录切换
- 设置按钮
- **部署按钮**（黑色主按钮）

### 2. 环境变量页面

#### 环境切换Tab
- **开发环境** (development)
- **生产环境** (production)

两个环境的变量互相独立，可以分别配置。

#### 环境变量列表

每个环境变量包含：

**Key字段**
- 显示变量名称
- 右侧复制按钮
- 最多63个字符
- 禁用状态（灰色背景）

**Value字段**
- 显示变量值
- 默认隐藏显示（••••••••）
- 右侧眼睛图标：点击切换显示/隐藏
- 禁用状态（灰色背景）

**操作菜单**
- 更多操作按钮（三点图标）
- 下拉菜单包含：
  - 删除选项（红色危险操作）

#### 新建变量表单

点击"新建变量"按钮后显示：

**Key输入框**
- 占位符："请输入Key，最多63个字符"
- 右侧复制图标（禁用状态）
- 最大长度限制：63字符

**Value输入框**
- 占位符："请输入Value"
- 右侧眼睛图标：可切换显示/隐藏输入内容

**操作按钮**
- 右侧更多按钮（禁用状态）

**底部按钮区**
- **取消**：取消添加，清空输入
- **确认**：保存环境变量（黑色按钮）

## 功能详细说明

### 1. 标签页切换

```typescript
// 标签页状态
const [activeTab, setActiveTab] = useState('preview');

// 点击标签切换页面
onClick={() => setActiveTab('env')}
```

### 2. 环境切换

```typescript
// 环境状态
const [envEnvironment, setEnvEnvironment] = useState<'development' | 'production'>('development');

// 两个环境的变量分别存储
const [devEnvVars, setDevEnvVars] = useState<EnvVariable[]>([]);
const [prodEnvVars, setProdEnvVars] = useState<EnvVariable[]>([]);
```

### 3. 添加环境变量

**交互流程：**
1. 点击"+ 新建变量"按钮
2. 显示输入表单
3. 填写Key和Value
4. 点击"确认"保存
5. 或点击"取消"放弃

**验证规则：**
- Key不能为空
- Value不能为空
- Key最多63个字符

### 4. 删除环境变量

**交互流程：**
1. 点击更多按钮（...）
2. 选择"删除"
3. 立即删除（无二次确认）

### 5. 显示/隐藏Value

**交互：**
- 点击眼睛图标切换
- 显示状态：眼睛睁开图标，显示明文
- 隐藏状态：眼睛闭上图标，显示 ••••••••

### 6. 复制Key

**交互：**
- 点击复制图标
- 复制Key到剪贴板
- 显示成功提示

## 数据结构

```typescript
interface EnvVariable {
  id: string;          // 唯一标识
  key: string;         // 变量名
  value: string;       // 变量值
  visible: boolean;    // 是否显示明文
}
```

## 核心函数

### getCurrentEnvVars()
获取当前环境的变量列表

### setCurrentEnvVars(vars)
设置当前环境的变量列表

### handleAddEnvVar()
添加新环境变量
- 验证输入
- 添加到当前环境
- 清空表单
- 显示成功提示

### handleDeleteEnvVar(id)
删除指定环境变量
- 从列表中移除
- 显示成功提示

### handleToggleEnvVisible(id)
切换变量值的显示/隐藏状态

### handleCancelAddEnvVar()
取消添加操作
- 关闭表单
- 清空输入
