# 智能体群组功能演示指南

## 快速开始

```bash
cd /Users/apple/Documents/skills/admin-system
npm start
```

访问 http://localhost:3000 查看前台页面

## 功能演示步骤

### 步骤1：新建群组

1. 打开前台页面
2. 在左侧边栏找到"新建群组"按钮（在"新建对话"按钮下方）
3. 点击按钮打开创建群组弹窗
4. （可选）输入群组名称，例如："技术支持团队"
5. 在树形选择器中选择智能体：
   - 展开"编程开发"类别
   - 选择"Python编程助手"和"JavaScript专家"
   - 展开"数据分析"类别
   - 选择"数据分析师"
6. 查看底部的智能体预览区域
7. 点击"创建"按钮

### 步骤2：查看群组列表

创建成功后，在左侧边栏会看到：
- "我的群组"模块出现在"智能体中心"下方
- 显示刚创建的群组卡片
- 卡片上显示：
  - 团队图标
  - 群组名称
  - 前3个成员的图标
  - 成员总数
- 右上角显示群组总数徽章

### 步骤3：进入群组对话

1. 点击群组卡片
2. 主界面切换到群组对话模式
3. 观察界面变化：
   - 顶部显示群组信息栏
   - 中间显示空的消息区域，带有提示信息
   - 底部显示输入框

### 步骤4：@智能体发送消息

#### 方式1：@单个或多个智能体

1. 在输入框中输入 `@` 符号
2. 自动弹出智能体选择面板
3. 点击"Python编程助手"
4. 输入框上方显示已选择的智能体标签
5. 可以再次输入 `@` 选择更多智能体
6. 在输入框中输入问题："请帮我分析这段代码的性能问题"
7. 点击"发送"按钮

#### 方式2：@all所有成员

1. 在输入框中输入 `@`
2. 点击面板顶部的"@all (提及所有成员)"选项
3. 所有群组成员都会被添加到已@列表
4. 输入消息并发送

### 步骤5：观察智能体回复

发送消息后，每个被@的智能体会依次回复，观察以下阶段：

#### 阶段1：Thinking
- 智能体头像出现
- 显示智能体名称
- 灰色气泡显示"正在分析问题..."
- 有加载动画

#### 阶段2：Tool Calls（约1秒后）
- 显示正在调用的工具
- 黄色背景的工具卡片
- 例如："web_search"、"read_file"
- 每个工具旁边有加载动画

#### 阶段3：Completed（约3秒后）
- 显示完整的回复内容
- 浅色背景的消息气泡
- 底部显示使用过的工具清单（绿色对勾）
- 完整的回复文本

### 步骤6：继续对话

1. 可以继续@不同的智能体
2. 可以移除已@的智能体标签（点击×号）
3. 发送更多消息
4. 观察多个智能体的回复

### 步骤7：退出群组

1. 点击顶部的"退出群组"按钮
2. 返回到欢迎界面
3. 群组对话记录会被清空

### 步骤8：创建更多群组

重复步骤1，创建不同主题的群组：
- "内容创作团队"：选择文案写作助手、翻译助手
- "行业分析团队"：选择互联网行业洞察、石油行业知识问答
- "全能团队"：选择多个类别的智能体

当群组超过5个时，会显示"查看全部"按钮

## 已知演示限制

1. **模拟数据**：智能体回复为固定的模拟内容
2. **无持久化**：刷新页面后群组和消息会丢失
3. **固定延时**：thinking/tool calls/completed的时间为固定值
4. **工具调用**：显示的工具为演示数据，非真实调用
5. **无错误处理**：未实现网络错误、超时等异常处理
6. **无消息历史**：退出群组后对话记录会清空
