# 多智能体协同Agent配置页面 - 使用说明

## 🎯 新增功能

### 创建了完整的多智能体协同Agent配置页面
文件位置：`src/pages/AgentCollaborative.tsx`

## 🚀 访问路径

### 完整操作流程：

```
1. 访问 http://localhost:3000/#agent
   ↓
2. 点击"创建Agent"按钮
   ↓
3. 在弹窗中选择"多智能体协同Agent"
   ↓
4. 进入多智能体协同配置页面
```

### 直接访问：
```
本地: http://localhost:3000/#agent-collaborative
局域网: http://10.253.214.98:3000/#agent-collaborative
```

### 二、左侧：应用配置面板

#### 1. 应用设置（可折叠）
- 占位功能，待扩展

#### 2. 基本信息
- **应用图标**：支持输入emoji，实时预览
- **应用名称**：限制50字，带字数统计
- **应用描述**：限制100字，支持AI生成按钮

#### 3. 开场白
- 多行文本输入
- 自适应高度
- 用于设置Agent的首次问候语

#### 4. 推荐问
- 支持添加最多5个推荐问题
- 每个问题可删除
- 输入框支持回车添加

### 三、中间：Agent配置面板

#### 🤖 主Agent配置

**功能**：
- 负责规划和拆解任务
- 分配任务给子Agent执行

**配置项**：

1. **模型选择**
   - DeepSeek-V3.1-250821
   - GPT-4
   - GPT-3.5
   - 支持刷新图标

2. **参考上文数**
   - 滑块控制（1-10）
   - 实时数字显示
   - 控制主Agent参考的历史对话轮数

3. **提示词**
   - 大文本框（最多1000字）
   - 带字数统计
   - 支持模板按钮
   - 用于定义任务拆解规则

#### 👥 子Agent管理

**功能**��
- 展示已添加的子Agent
- 支持启用/禁用
- 支持删除
- 限制最多10个

**操作**：

1. **添加子Agent**
   ```
   点击"+ 添加 (2/10)"按钮
   ↓
   弹出子Agent选择弹窗
   ↓
   浏览可用的官方/自定义Agent
   ↓
   点击"添加"按钮
   ↓
   Agent添加到列表
   ```

2. **已添加的子Agent卡片**
   - 显示图标、名称、描述
   - "官方"标签
   - 启用/禁用开关
   - 删除按钮

3. **预置的子Agent**
   - 🌐 网页探索Agent：操作网页，主动探索
   - 💻 代码编写Agent：生成并运行Python代码
   - 📊 数据分析Agent：统计分析和可视化
   - ✍️ 文案创作Agent：多种文案类型
   - 🎨 图像生成Agent：文本生成图像
   - 📚 知识问答Agent：基于知识库问答

### 四、右侧：预览与调试面板

**功能**：
- 应用图标预览（大尺寸80px）
- 应用名称展示
- 聊天输入框（模拟用户交互）
- 使用限制提示文字

**交互元素**：
- 输入框支持emoji按钮
- 发送按钮（蓝色图标）
- 显示每日使用次数

### 五、顶部操作栏

**按钮功能**：
- ⬅ 返回：返回Agent列表
- 💾 配置：保存配置
- 👁 发布：预览应用
- 📄 文件：文件管理
- 💾 保存：保存草稿
- 🚀 发布：发布应用

## 🎨 交互特性

### 1. 添加子Agent弹窗

**触发**：点击"+ 添加"按钮

**界面**：
```
┌─────────────────────────────────────┐
│  添加子Agent                    [×] │
├─────────────────────────────────────┤
│  [搜索Agent...]                     │
│                                     │
│  ┌───────────────────────────┐     │
│  │ 📊 数据分析Agent  [官方]  │     │
│  │ 专业的数据分析能力...     │     │
│  │                  [+ 添加] │     │
│  └───────────────────────────┘     │
│                                     │
│  ┌───────────────────────────┐     │
│  │ ✍️ 文案创作Agent  [官方]  │     │
│  │ 专业的文案撰写能力...     │     │
│  │                  [+ 添加] │     │
│  └───────────────────────────┘     │
│                                     │
│  ... 更多Agent ...                 │
└─────────────────────────────────────┘
```

**特性**：
- 支持搜索过滤
- 显示Agent图标、名称、描述
- "官方"标签
- 已添加的显示"已添加"标签，禁止重复添加
- 点击卡片或"添加"按钮添加
- 添加后自动显示成功提示

### 2. 子Agent卡片交互

**悬停效果**：
- 边框变为蓝色
- 轻微阴影效果

**操作按钮**：
- 🗑 删除：移除该子Agent
- 🔘 开关：启用/禁用该子Agent

**状态区分**：
- 启用状态：正常背景色
- 禁用状态：灰色背景（暂未实现）

### 3. 推荐问题管理

**添加方式**：
- 输入框输入后点击"+"图标
- 或按回车键

**删除方式**：
- 点击问题项右侧的删除图标

**限制**：
- 最多5个问题
- 达到上限后禁止添加

### 4. 上下文数滑块

**交互**：
- 拖动滑块调整数值（1-10）
- 右侧实时显示当前值
- 影响主Agent的记忆能力


### 2. 动态添加/删除
```typescript
// 添加子Agent
const handleAddSubAgent = (agent: SubAgent) => {
  if (!subAgents.find(a => a.id === agent.id)) {
    setSubAgents([...subAgents, { ...agent, enabled: true }]);
  }
};

// 删除子Agent
const handleRemoveSubAgent = (agentId: string) => {
  setSubAgents(subAgents.filter(a => a.id !== agentId));
};
```
